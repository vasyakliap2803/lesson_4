Властивості CSS для тексту

color - задає колір тексту. Підтримує ключові слова, шістнадцяткові (HEX), RGB та HSL-формати.

Це речення буде фіолетовим.

font-family - визначає шрифт для тексту. Список шрифтів задається через кому: перший доступний шрифт використовується.

Це речення буде Courier New, або Courier, якщо Courier New недоступний.

font-size - задає розмір шрифту. Може бути в пікселях (px), відносних одиницях (rem, %) або абсолютних величинах.

Це речення буде розміром 24px.

font-weight - контролює товщину шрифту. Можливі значення:

normal або 400: звичайний текст.

Це речення буде нормальною товщини.

bold або 700: жирний текст.

Це речення буде жирним.

lighter: тонший шрифт порівняно з батьківським елементом.

Це речення буде мати тонший шрифт порівняно з батьківським елементом.

bolder: товщий шрифт порівняно з батьківським елементом.

Це речення буде мати товщий шрифт порівняно з батьківським елементом.

Числові значення: від 100 (тонкий) до 900 (дуже жирний).

Це речення буде мати значення font-weight: 100.

Це речення буде мати значення font-weight: 200.

Це речення буде мати значення font-weight: 300.

Це речення буде мати значення font-weight: 400.

Це речення буде мати значення font-weight: 500.

Це речення буде мати значення font-weight: 600.

Це речення буде мати значення font-weight: 700.

Це речення буде мати значення font-weight: 800.

Це речення буде мати значення font-weight: 900.

Інші важливі властивості для тексту

font-style - задає стиль шрифту, наприклад, курсив.

normal: звичайний текст без стилю.

italic: курсив.

oblique: похилий текст, подібний до курсиву.

line-height - визначає висоту рядка. Може бути:

Числове значення (1.5): відносна висота рядка до розміру шрифту.

Відсотки (150%): висота рядка у відсотках від розміру шрифту.

Фіксовані одиниці (наприклад, px): абсолютна висота рядка.

Це речення матиме міжрядковий інтервал 2.

letter-spacing - задає відстань між літерами.

Позитивне значення (2px): збільшує відстань між літерами.

Негативне значення (-1px): зменшує відстань між літерами.

text-align - вирівнює текст всередині блочного елемента. Можливі значення:

left: вирівнює текст по лівому краю (за замовчуванням для ліво-направлених мов).

right: вирівнює текст по правому краю.

center: вирівнює текст по центру.

justify: вирівнює текст по обидва боки, додаючи пробіли між словами для вирівнювання рядків.

text-decoration - додає до тексту підкреслення, накреслення зверху або перекреслення. Значення:

underline: підкреслює текст.

overline: додає лінію над текстом.

line-through: перекреслює текст.

none: видаляє будь-які стилі.

text-transform - змінює регістр (регістр) тексту. Значення:

uppercase: переводить весь текст у верхній регістр.

LOWERCASE: переводить весь текст у нижній регістр.

capitalize: робить перші літери кожного слова великими.

none: залишає текст без змін.

text-indent - задає відступ для першого рядка тексту в елементі. Може бути у відсотках або фіксованих одиницях.

Це речення матиме відступ для першого рядка.

word-spacing - контролює відстань між словами. Позитивні значення збільшують відстань, а негативні зменшують.

Це речення матиме відстань між словами 20px.

Це речення матиме відстань між словами -3px.

Призначення кольорів у CSS

Ключові слова для кольорів

CSS підтримує стандартні назви кольорів, такі як red, blue, green, black і т. д. Це речення буде синім.

Шістнадцятковий формат (HEX)

HEX-код складається з шести символів, які представляють червоний, зелений і синій канали кольору (RGB). Наприклад, #FF5733. Це речення буде помаранчевим.

Формат RGB і RGBA

Формат RGB задає колір через значення червоного, зеленого і синього каналів у діапазоні від 0 до 255. RGBA додає прозорість через четвертий параметр (альфа-канал). Це речення буде напівпрозорим червоним.

HSL та HSLA

HSL використовує три параметри для задання кольору: тон (hue), насиченість (saturation) і яскравість (lightness). HSLA додає прозорість. Це речення буде яскраво-зеленим.

Тіні в CSS

text-shadow Задає тінь для тексту. Синтаксис:

Перший параметр: горизонтальне зміщення.

Другий параметр: вертикальне зміщення.

Третій параметр (необов'язковий): розмиття.

Четвертий параметр (необов'язковий): колір тіні.
